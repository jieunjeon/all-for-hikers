# All For Hikers
Django Backend for 'All For Hikers' app


## Stack
- Python
- Django
- Tailwind CSS
- Gulp
- ES6
- Mailgun

## Features
### Users
- Kakao Talk Login
- Github Login
- Admin Panel
- Admin Mode
- DM (Direct Messages)

### Hiking Routes
- Hiking Route CRUD
- Image Upload
- Review System

## Deployment
- AWS EB
- AWS S3
- AWS RDS
