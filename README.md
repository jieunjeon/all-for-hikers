# All For Hikers
Django Backend for 'All For Hikers' app


## Stack
- Python
- Django
- Django Rest Framwork
- Django Graphene
- Tailwind CSS
- GraphQL API
- Redux
- Gulp
- ES6
- Mailgun
- Docker
- AWS


## Features
### Users
- Kakao Talk Login
- Github Login
- Admin Panel
- Admin Mode
- DM (Direct Messages)
- Favorites

### Hiking Routes
- Hiking Route CRUD
- Image Upload
- Review System
- Route Search
- Map Screen

## Deployment
- AWS EB
- AWS S3
- AWS RDS
- Docker
